Донт андэстэнд

6 декабря 2010 г.

Сегодня один из заказчиков попросил заменить документы в нескольких разделах его сайта. Разделов с документами 2, в каждом примерно по 7 документов.

Предвидя возможные траблы попросил «отправить два письма, в каждом из которых написано название раздела и вложены документы предназначенные для размещения в данном разделе.

Получаю почту. 4 письма

  • ДОКУМЕНТЫ ДЛЯ ВСТУПЛЕНИЯ
  • ДОКУМЕНТЫ ДЛЯ РАЗДЕЛА С ЗАЩИЩЕННЫМ ДОСТУПОМ
  • ЕЩЁ ДЛЯ ЗАЩИЩЁННОГО ДОСТУПА (ДЛЯ ЗАМЕНЫ)
  • ЕЩЕ ДЛЯ ВСТУПЛЕНИЯ
  • Максим, обратите внимание, что в документах для вступления отсутствовало заявление. Я отправила Вам его раньше в предыдущем письме
