Моя подборка программ для Mac

4 ноября 2013 г.

Уже больше года, как у меня дома появился IMac, а чуть позже к нему добавился и Macbook. Работать на компьютерах Apple одно удовольствие, удобных (и платных) программ хватает буквально на каждый чих.

Пройдёмся по списку:

Google Chrome — браузер, который установлен на всех моих компьютерах. Я в курсе, что тру-программисты предпочитают Firefox, но я привык именно к хрому. Синхронизация позволяет мне открывать на телефоне вкладки, открытые на компьютере, плюс я даже пару раз использовал функцию печати на облачный принтер. Установлена дополнения для работы с куками, сохранения сессий, баннерорезалка, stylebot.

После регистрации в Dropbox бесплатны 2Gb, но даже этого более чем достаточно для синхронизации документов находящихся в работе и настроек многих программ

DropBox — одна из первых программ, которые я устанавливаю на новый компьютер. Де-факто — стандарт облака для пользователя.

Adium — ICQ-клиент. В последнее время пользуюсь ICQ для связи буквально с двумя-тремя людьми.

Sublime Text — редактор кода. Есть ещё купленный PHPStorm, но прижился именно легковесный Sublime Text. Нравится цветовая схема и автодополнение. Пользоваться полноценным IDE нет ни нужды, ни образования.

iTerm — замена стандартного терминала. Вкладки, закладки. Когда нужно зайти на сервер по SSH.

Transmit ($34) — работа с файлами на сервере по протоколу sFTP. Поддерживает Amazon S3, умеет синхронизировать закладки через Dropbox.

SequelPro — работа с базами данных. Поддерживает тунеллирование, закладки.

Какие преимущества у популярного под Mac 1Password, который стоит от 50 долларов, я так и не понял

KeePassX — аналог KeePass под Windows. Хранилище паролей. База паролей шифруется мастер-паролем, сама база синхронизируется через Dropbox.

Evernote ($7 в месяц) — используется для хранения всякой не первой важности информации. Платный аккаунт для возможности хранить блокноты оффлайн и шарить их с другими пользователями.

iA Writer — markdown-редактор. Использую иногда для написания текстов в блог.

nvAlt — форк от популярного сервиса для хранения информации Notational Velocity. Поддерживает markdown-разметку. Мгновенный поиск, возможность хранения информации в файлах, сами файлы лежат в папке Dropbox.

PopClip — широкие возможности работы с выделенным текстом. При выделении появляется всплывающее меню, в котором есть куча кнопок настраиваемых плагинами. Чаще всего я использую его в связке с Translate Tab.

Translate Tab — программа висящая в трее, позволяет переводить текст через Google Translate без браузера.

PuntoSwitcher — исправление неправильной раскладки клавиатуры. Как можно пользоваться компьютером без этой программы я не знаю.

TextExpander ($34.95) — использование сокращений при наборе текста. Так, если мне нужно набрать номер кошелька WebMoney я просто наберу WMR# это сокращение исправить набранное на R346263669864 (не забудь отправить сюда 100 рублей, как обещал).

В строке Alfred2, кроме этого, можно производить арифметические действия, а с купленным PowerPack создавать целые workflows. К примеру набрав 'cur 10$ RUB' я получу стоимость 10 долларов в рублях по версии Google Finance.

Alfred2 + PowerPack (£ 17) — расширенный лаунчер для Mac. Я уже давно не нажимаю кнопку «Пуск» и не занимаюсь путешествием по меню, если хочу открыть тот же Sublime Text. Нажимаю Alt+Пробел на экране появляется строка для набора. Начинаем печатать, двух букв «Su…» хватает, чтобы запустить именно Sublime Text.

Little Snitch (€ 59) — файрволл для Mac. В бесплатной версии работает два часа после перезапуска. У меня платная.

uTorrent — качалка торрентов. Ровно то же, что и на Windows.

Bartender ($15) — программа, которая прячет редко используемые программы в трее. В Windows это можно делать бесплатно.

Resizeit — простой работающий пакетный ресайз картинок

Anki — программа для заучивания английских слов

TweetDeck — клиент для твиттера. Использую, когда нужно что-то написать от аккаунта @oldkemerovo, чтобы не перелогиниваться в браузере.

Teamviewer — работа с удалёнными рабочими столами

Palua ($0.99) — по умолчанию кнопки F1-F12 на Mac работают вовсем не так как в Windows — регулировка громкости, яркости дисплея, управление плеером. Иногда нужны именно кнопки F1-F12. Программа Palua позволяет задать правила работы этих кнопок для конкретных программ.

Hazel ($28) — беспорядок в папке «Загрузки» беспокоит ежедневно тысячи людей. На Mac я для себя эту проблему решил с помощью программы Hazel и 28 долларов. Программа позволяет настраивать практически любые правила для работы с файлами в любых папках. У меня она просто удаляет из папки «Загрузки» файлы старше 3 дней, а файлам старше суток присваивает тег «Серый».

Monosnap ($0.99) — удобная программа для управления скриншотами. Позволяет рисовать стрелочки, линии, подписывать комментарии. По умолчанию хранит скриншоты на своём сервере. Я, скорее в качестве благодарности за софт, купил дополнение, которе позволяет выкладывать скриншоты в Dropbox (там ещё много вариантов от Amazon S3 до Яндекс.Диска и собственного сервера).
