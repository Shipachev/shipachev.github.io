Луна — не спутник Земли #

Несмотря на кажущуюся глупость такого заявления в нём есть немалая доля правды. Со школы мы знаем, что Земля вращается вокруг Солнца, а Луна вращается вокруг Земли. На самом деле всё выглядит чуть иначе. С точки зрения гравитации, Луна — не спутник Земли, а одна из планет системы планет Земля-Луна.

Луна отличается от других спутников планет Солнечной системы. Диаметр Луны всего в 3,6 раза меньше диаметра Земли. Для сравнения диаметр Ганимеда (самого большого спутника Юпитера) меньше самого Юпитера в 28 раз. Масса Луны в 81 раз меньше массы Земли (Ганимед имеет массу в 12000 раз меньшую массы своей планеты).

На орбите Луну удерживает не притяжение Земли (в большей степени), а, как и другие планеты Солнечной системы, притяжение Солнца. Если так получится, что Земля остановится на месте, Луна продолжит своё вращение вокруг Солнца.

Ass-tag convention #

Негласная договорённость по которой та сторона полотенца, на которой этикетка, предназначена для той части тела, что ниже пояса, и наоборот. Пригодится, если вдруг вам придётся использовать одно полотенце на двоих с не совсем знакомым человеком.

Крипта цивилизации #

25 мая 1940 года в Оглторпском университете города Брукхейвен, штат Джорджия, США была сконструирована и запечатана так называемая крипта цивилизации — герметично закрытое помещение, содержимое которого предназначено для наших потомков. В крипте хранятся более 800 книг, записанных на ацетилцеллюлозной плёнке, записи голосов Адольфа Гитлера, Иосифа Сталина, Бенито Муссолини и Франклина Рузвельта, голос моряка Попая и «свиной манок». Из технических устройств — устройство для чтения микрофильмов, проектор для изображений, тостер, радио, генератор, работающий от текущей воды, для запуска всех этих устройств, пишущую машинку, кассовый аппарат. Кроме этого образцы семян, зубная нить, содержимое дамской сумочки, соска, специальным образом разлитая банка пива Budweiser.

Группы крови у японцев #

Многие помнят истерию по поводу гороскопов в середине 90-х годов. Несмотря на «восточность» этих гороскопов, в Стране восходящего солнца с теми же чувствами относятся не к гороскопу, а к группам крови человека (и непонятно, кто ещё прав). Группу крови указывают в службах знакомств и при приёме на работу. В японском фейсбуке также есть такая возможность.

Условие Йоды #

Стиль записи при котором значение переменной функции сравнения записывается перед переменной называется Условием Йоды (по имени персонажа из «Звёздных войн»).

обычная запись

if ( $var == 42 ) { … }

условие Йоды

if ( 42 == $var ) { … }

Данный стиль записи защищает от ошибки, при которой происходит не сравнение, а присваивание переменной ($var = 42), но, на мой взгляд, значительно ухудшает читабельность кода.
